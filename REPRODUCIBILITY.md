# Reproducibility Instructions

To reproduce the archived workflow:

1. Install the required Python packages listed in `requirements.txt`.
2. Place the cleaned source datasets in `data/raw_cleaned/`.
3. Run the Python scripts in `code/`.
4. Review the processed datasets in `data/processed/`.
5. Review the output tables in `outputs/tables/` and `outputs/LTC_Reproducibility_Outputs.xlsx`.

The output tables correspond to the analyses reported in the associated study.
